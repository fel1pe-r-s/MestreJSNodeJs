#React #react #hook como o componente renderiza sempre que sofre uma alteração

o useEffect permite executa uma função no primeiro parâmetro, quando um determinado valor é alterado no seu segundo parâmetro ou quando o componente é renderizado a primeira vez.
exemplo uma requisição a api pode ser feita usando o useEffect assim na primeira vez que o componente for renderizado ele executa a função para chamada a api.
