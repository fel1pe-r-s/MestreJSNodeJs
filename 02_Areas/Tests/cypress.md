#Tests

[Doc](https://docs.cypress.io/guides/getting-started/installing-cypress)
É uma lib para tests E2E no front end outra alternativa seria o playwright 
Ele vai criar um test que vai se comporta como um usuário acessando nossa aplicação.

o Cypress pode ser usado também para tests de components

Já os tests E2E não dependendo da tecnologia estamos usando.

Podemos roda os test via linha de comando ou pela interface.
